ğŸ“± How to View & Test Your App

Option 1: Test on Your Phone (Recommended)

  1. Download "Expo Go" app from App Store (iOS) or Play Store (Android)
  2. Make sure your phone and computer are on the same WiFi network
  3. In your terminal, run: cd /app && npm run dev
  4. Scan the QR code that appears with your phone camera (iOS) or Expo Go app (Android)
  5. The app will open in Expo Go on your phone

Option 2: View Code in Your Editor

  1. Open your code editor (VS Code, Cursor, etc.)
  2. Navigate to the /app directory
  3. Key files to explore:
    • /app/app/auth/login.tsx - Login screen
    • /app/app/(tabs)/index.tsx - Home feed
    • /app/lib/auth-context.tsx - Authentication logic
    • /app/components/StartupCard.tsx - Startup card component

Option 3: Build for Production

  1. Install EAS CLI: npm install -g eas-cli
  2. Login: eas login
  3. Build for iOS: eas build --platform ios --profile production
  4. Build for Android: eas build --platform android --profile production
